## Responsive Design

### Desktop (width >= 640px)
- Full text display
- Icon + text + number in one line
- Gradient background visible
- Placed in left sidebar

### Mobile (width < 640px)
- Same appearance in mobile sidebar
- Text may wrap on very small screens
- Accessed via hamburger menu
- Still displays above chat list
